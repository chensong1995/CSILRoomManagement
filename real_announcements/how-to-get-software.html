How to get software for CMPT courses

We provide both Windows and Linux compatible software in a number of software repositories for students. We do not provide Apple brand software at this time. Microsoft brand software used in CSIL is licensed through the Microsoft Developer Network Academic Alliance (MSDNAA) program, (aka "Microsoft Imagine"). On this page, we use MSDNAA to refer this program.

Since we could not assist on any downloading issues with the non SFU sites, please NEVER use download option. Always come to CSIL and get/burn a copy!

The access to these software packages will be available within the same time period while the CSIL is open.

Who can get CMPT course software?

Every student who is registered in any CMPT course in the current semester can get copies of any software packages we have available in our repositories. We make this software available to CMPT students so that they can do a portion of their course work using their own computer if they wish.

Please note: if a student takes MACM101 but no CMPT courses, this user has no access to MSDNAA software repository.

What format does the software come in?

The software comes in a number of different formats. All Linux and some Microsoft brand and Windows compatible software comes in the form of ISO images. Some Microsoft brand and Windows compatible software comes in the form of installers (*.msi), updaters (*.msu), and executables (*.exe). Additional formats may be found in the repositories for different software.

How to use the ISO files?

The ISO files could not be used directly; there are 2 ways to access the content within:

  1.  Burn the ISO file to a writable & blank CD/DVD disk. On CSIL Windows workstations, you could use CDBurnerXP to do this.
  2.  If you do not want to burn a disk, or there is no optical drive on your computer, you need to use a CD/DVD emulation software. You may want to read this external link to make your selection.

What is the procedure for getting software?

IMPORTANT: For any MSDNAA product, please "purchase" it before getting the media. And always copy the licensing information for your purchase (e.g. product keys) to at least two of your own places.

We recommend that general users simply burn their selected ISO files to a CD or DVD. You can then read your CD/DVD on your computer and install your chosen software just as if it were an original installation CD/DVD. You can burn DVDs/CDs in CSIL using the installed CDBurnerXP software. If you choose to copy ISO files themselves to your own storage, please use a CD/DVD emulation software to access the content within.

For our non-ISO format software packages, simply copy them to any removable storage (CD/DVD, USB key or USB external hard drive) and take them home.

Burnaby Campus Users and Surrey campus users

From any CSIL@Burnaby Windows workstation you can access the CSIL software repository via the M: drive.

The repository currently has 3 sections/folders:

. Linux: Linux Distros (selected)

. Misc: Software installed on CSIL Windows systems (partial)

. MSDNAA: MSDNAA products (selected)

In CSIL@Burnaby, every workstation is equipped with a DVD burner.

The software for burning ISO images is CDBurnerXP.

In CDBurnerXP, select "Write image file to disc", then browse to the M: drive and find the software you want to burn.

Each application we make available in the repository has its own licensing scheme. Please read the end-user agreement carefully during the installation.

These packages are also available for Windows systems in TASC1 9401 (the Common Room). The tool for burning CD/DVD is CDBurnerXP. After logon to a Windows system in this room, simply access the repository

\\cs-gaia.cmpt.sfu.ca\repository

directly in your Windows Explorer (the Windows File Manager, not the browser Internet Explorer)

About MSDNAA/Microsoft Imagine products

 

Microsoft requires students to purchase each Microsoft brand product via the E-Academy web site. Because the School of Computing Science is a member of the MSDNAA, student's purchases are FREE ($0.00) if you don't require the media (CD/DVD).

Note: Any license agreements you commit to are strictly between you and Microsoft.

Please visit our service page which will link to the ELMS for Computing Science at SFU site. You will need your SFU campus ID and password to login to this site. Once logged on, select the software you want under download section.

The School's subscription is: "Simon Fraser University - CS - Microsoft Imagine Premium". If you browse this site to locate our subscription, please make sure this is the store.

After you check out, you will get a purchase receipt, and if it is required for your chosen product, a product key. Note that some products already contain the product key in the distribution files, but Microsoft still requires students to obtain receipts for their software.

Once you have your product receipt (and product keys if provided), you can burn your own copy of the software in CSIL as mentioned above, saving the downloading time/bandwidth it takes to download from the MSDNAA site.

Note that there are many reports claiming the downloading process from this site is not stable. And there are reports indicate that the "Mail Order" does not reach satisfactory level either.

So, please:

do NOT download products from this web site

do NOT purchase "Mail Order"

You are on your own if you choose these options - there is nothing helpdesk of the School could help you out.

WARNING: if you have graduated and/or no longer taking CMPT courses, you will no longer be able to get the keys from the web site, nor from the School.

So, please keep your copies of MSDNAA product safe and accessible, keep your purchased keys safe and accessible. Important: the information on MSDNAA site is NOT considered accessible!

Where to save the licensing information of my purchases so I could access it later?

Once you've purchased any MSDNAA product, there will be a record on the MSDNAA site. The tricky part is: the information on MSDNAA site is NOT considered accessible!

If you have graduated and/or no longer taking CMPT courses, YOU WILL NO LONGER BE ABLE TO GET THE KEYS FROM THIS WEB SITE, NOR FROM THE SCHOOL.

So, if you would like to have access to these crucial information at any occasions, please follow these steps:

  • click "Your Accounts/Orders"
  • under the "Order History", click each order
  • (1) print page (with product keys etc.) as a PDF file
  • (2.0) email this PDF file to your SFU Connect account.
  • (2.1) email this PDF file to your secondary email account.
  • (2.2) print this PDF file to paper, and keep this piece of paper safe with the product media, e.g. the DVD you just burnt in CSIL.

Please keep your copies of MSDNAA product safe and accessible, keep your purchased keys safe and accessible.

I see there is this "Access Guarantee" option on the MSDNAA site. Does this mean I will have access to my previous purchase records after graduation or not taking any CMPT courses?

No.

This is an extremely misleading piece of information.

It basically means: if (and only if) you have access to the MSDNAA site, you have access to your previous purchase records in certain period of time.

If you no longer have access to the MSDNAA site, you will have no access to your previous purchases. PERIOD.

BTW, do not purchase the "Extended Access Guarantee" - it is purely wasting money.

So, please keep the record in your own domain; please check the previous Q&A for more details.

I could not activate a MSDNAA product with the key I received, what I could do?

As we have stated on this page, we (helpdesk of the School) could not help you. You have to contact Microsoft all by yourself to get the product activated.

e.g. If the activation of Windows 7 fails, you may try to activate it manually:

  • run command "slui.exe 4"
  • follow the instruction, select "Phone Activation" option.
  • call the number listed and hold on for a real person - this may take a while.

How long I could keep using the MSDNAA products?

The license for any MSDNAA product doesn't expire as long as you conform to the usage guideline and the EULA, which essentially states that you are using it for your course work and not for commercial work (EULA clause 5a).

Please read more: the Usage Guideline, and the EULA.

I could not login to purchase VMWare products, what I should do?

Some people have reported the issue that they are taking CMPT courses but could not login to OnTheHub Network to purchase VMWare products - but they could login to purchase MSDNAA products!

We are investigating this issue.

In the meantime, you may try the following:

Visit the "Simon Fraser University - School of Computing Science" webstore page on the OnTheHub Network.

Click "Sign in" and then click the link "Forgot username or password?" under the button "Sign in".

Key in your SFU Connect email address (not alias!), submit the request.

Wait for a few minutes and check your email.

Locate the email with subject line "Your username from Simon Fraser University - School of Computing Science" - it contains the instructions on reset your credentials to purchase VMWare products.

Follow the instructions and reset your password and you should be able to login

General Tips

  • Keep the product key and the media in safe place.
    • Please always generate a PDF file of your purchase for records.
  • If your software requires a Service Pack, get it at the same time. Install the Service Pack asap.
  • On your Windows system(s), install "Microsoft Updates" to obtain latest updates for all Microsoft products.
  • Use the lowest possible speed when burning CD/DVD disks to ensure the highest quality.

What if the software I need is not on the list?

There are many reasons why a specific application is not on the list:

  • The software needs a special license, e.g. Matlab.
  • The software has compatibility issues in CSIL, e.g. Adobe Reader 9.0.
  • The software is not related to CMPT studies.
  • This is not a popular package so we did not put in the repository due to storage space limitations.

Please email helpdesk if you need other software packages.
